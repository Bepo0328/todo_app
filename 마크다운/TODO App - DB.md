# 1. DB
	- 앱 사용에 필요한 데이터를 저장하고 관리할 수 있는 기능

	- 테이블
    데이터를 저장하는 표(구조)
    
|title|color|memo|
|---|---|---|
|책 읽기|빨강|파피용 읽기|
|강의 수강|하늘|유튜브 강의|
	
	- Column(데이터 카테고리) 예: title, color, memo
	- Row(데이터) 예: 책읽기, 빨강, 파피용 읽기
